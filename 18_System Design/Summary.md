## Perbedaan SQL dan NoSQL
- SQL merupakan sebuah relational database yang mempunyai entity yang saling berhubungan dan terstruktur. NoSQL tidak terstruktur dan tidak memiliki relasi antar collection
- Benefit Relational DB : Dirancang untuk segala keperluan, memiliki standar yang jelas dan memiliki banyak tools
- Kelebihan dari NoSQL -> schema less, fast development 
## Caching
- Merupakan sebuah penyimpanan data sementara untuk menampung data request yang sama sehingga ketika terjadi pemanggilan data yang sama tidak akan lama karena data tersebut sudah ditampung di cache
